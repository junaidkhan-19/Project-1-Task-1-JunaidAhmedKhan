# 🛍️ Customer Segmentation Using Unsupervised Learning

## 📌 Project Objective

The objective of this project is to segment mall customers based on their purchasing behavior using **K-Means Clustering**. By identifying distinct customer groups, businesses can develop targeted marketing strategies to improve customer engagement, retention, and overall sales performance.

## 📊 Approach

* Loaded and explored the **Mall Customers Dataset** using Pandas.
* Performed **Exploratory Data Analysis (EDA)** to understand customer demographics and spending patterns.
* Checked for missing values and duplicate records.
* Standardized numerical features using **StandardScaler**.
* Applied the **Elbow Method** to determine the optimal number of clusters.
* Implemented **K-Means Clustering** to group customers into meaningful segments.
* Visualized customer clusters using **PCA** and **t-SNE** for dimensionality reduction.
* Proposed marketing strategies tailored to each customer segment.

## 📈 Results & Insights

* Successfully identified **five distinct customer segments** based on annual income and spending score.
* PCA and t-SNE visualizations provided clear separation between customer groups.
* The analysis highlighted high-value customers, potential customers, and budget-conscious shoppers, enabling data-driven marketing decisions.
* The project demonstrates how **unsupervised machine learning** can be used to uncover hidden customer patterns and support strategic business planning.
